
Cài đặt

1.Clone project
2.Cài đặt packages:

npm install

3. Tạo file `.env`:

PORT=5000
MONGODB_URI=mongodb+srv://username:password@cluster.mongodb.net/al-shop

mn thay username:passworld --- quangtrung:quangtrung123

4. Chạy server:

npm run dev
